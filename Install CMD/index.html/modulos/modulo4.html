Módulo 4: LibreOffice Calc

Soluciones de análisis de datos en entornos de código abierto

🔎 Problema real

No todas las organizaciones tienen acceso a software de pago como Excel. En muchos casos, se requiere una alternativa gratuita que permita trabajar con datos de manera eficiente.

  • Limitaciones de licencias
  • Restricciones presupuestarias
  • Necesidad de soluciones libres

🧠 ¿Qué es LibreOffice Calc?

LibreOffice Calc es una herramienta de hoja de cálculo de código abierto que permite analizar, estructurar y manipular datos sin depender de software propietario.

  • Gratuito y de código abierto
  • Compatible con archivos de Excel (.xlsx)
  • Funciones avanzadas de análisis

⚙️ Funcionalidades clave

  • Funciones similares a Excel (BUSCARV, SI, SUMA, etc.)
  • Filtros y ordenamiento de datos
  • Tablas dinámicas
  • Importación y exportación de datos

💡 Ejemplo aplicado

Escenario: Tienes una base de datos de clientes y necesitas clasificarla.

Solución:

  • Usar filtros para segmentar información
  • Aplicar funciones para categorizar datos
  • Crear tablas dinámicas para resumir información
=SI(A2>100, "Cliente Premium", "Cliente Regular")

⚖️ Comparación con Excel y Google Sheets

  • Más ligero que Excel en equipos con pocos recursos
  • No requiere conexión a internet (a diferencia de Google Sheets)
  • Menos automatización avanzada que Power Query

🧪 Ejercicio práctico

Realiza lo siguiente:

  • Importa un archivo Excel en LibreOffice Calc
  • Limpia los datos (duplicados, formatos)
  • Crea una tabla dinámica
  • Clasifica datos usando funciones

Objetivo: Analizar datos sin depender de software de pago.
