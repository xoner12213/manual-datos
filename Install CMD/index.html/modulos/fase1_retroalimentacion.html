    • ¿Por qué plasmamos la información en el Bloc de Notas?

  • Texto Plano y Puro:

    El Bloc de Notas es un editor de texto sin formato. Al copiar datos de una web, un chat, un PDF o un correo, estos suelen arrastrar "basura invisible" (códigos HTML, formatos de fuente, colores, tabulaciones raras). El Bloc de Notas elimina toda esa basura inmediatamente.

  • Estructura CSV Real:

    Al pegar los datos allí y guardarlos con la extensión .csv (Valores Separados por Comas), obligamos al sistema a generar un archivo de base de datos en su estado más primitivo y limpio.

  • Estabilidad para Power Query:

    Power Query es una herramienta muy potente, pero es estricta. Si intentas cargar un Excel mal formateado, a veces intenta "adivinar" los formatos de celda y se equivoca. Al entregarle archivos CSV puros desde el Bloc de Notas, le facilitamos el trabajo de lectura y evitamos que columnas enteras aparezcan con errores (Error) o valores nulos (Null). Usar el Bloc de Notas fue nuestro filtro purificador para garantizar que el modelo de datos no fallara al primer intento.
